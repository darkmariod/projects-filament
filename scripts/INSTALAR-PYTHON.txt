AGENTE DE IMPRESION ZEBRA — INSTRUCCIONES
==========================================

PASO 1 — Instalar Python
-------------------------
1. Abrir el navegador e ingresar a: https://www.python.org/downloads/
2. Hacer clic en "Download Python" (el boton amarillo grande)
3. Ejecutar el instalador
4. MUY IMPORTANTE: marcar la opcion "Add Python to PATH" antes de instalar
5. Hacer clic en "Install Now"

PASO 2 — Instalar dependencias
--------------------------------
1. Presionar Windows + R, escribir cmd y presionar Enter
2. Pegar el siguiente comando y presionar Enter:

   pip install requests pywin32

3. Esperar que termine (tarda 1 o 2 minutos)

PASO 3 — Configurar la impresora
----------------------------------
1. Abrir el archivo agente.py con el Bloc de Notas
2. Buscar la linea que dice:
   PRINTER_NAME = "ZDesigner ZT411-203dpi ZPL"
3. Cambiar ese nombre por el nombre exacto de la impresora Zebra
   (para ver el nombre: Menu Inicio > Impresoras y escaneres)
4. Guardar el archivo

   Si la impresora Zebra tiene IP en la red, buscar la linea:
   PRINTER_IP = ""
   Y colocar la IP correspondiente, por ejemplo:
   PRINTER_IP = "192.168.1.200"

PASO 4 — Ejecutar el agente
-----------------------------
1. Abrir cmd (Windows + R, escribir cmd, presionar Enter)
2. Escribir lo siguiente y presionar Enter:
   python C:\Users\USER\Desktop\Agente\agente.py
   (ajustar la ruta segun donde se haya guardado el archivo)
3. Si todo esta correcto, se debe ver el mensaje:
   [OK] Servidor conectado. Agente activo

PASO 5 — Probar la impresion
------------------------------
1. Ingresar al sistema web
2. Generar un lote de etiquetas y enviarlo a imprimir
3. En la ventana del agente se debe ver:
   [OK] Item #1 impreso y reportado

Si aparece ese mensaje, el sistema esta funcionando correctamente.

PROBLEMAS COMUNES
-----------------
- "pip no se reconoce"       -> No se marco "Add Python to PATH". Reinstalar Python.
- "No se pudo conectar"      -> Verificar conexion a internet y que el sistema este activo.
- "Error USB"                -> Verificar que el nombre de la impresora sea exactamente igual.
